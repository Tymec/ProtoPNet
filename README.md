# ProtoPNet

Simple frontend for [ProtoPNet](https://github.com/cfchen-duke/ProtoPNet) to classify images uploaded by users.

## Installation
1. `git clone <repo_url> --recurse-submodules -j8`
2. `cd ProtoPNet`
3. `make install`
5. `make backend` and `make frontend`
